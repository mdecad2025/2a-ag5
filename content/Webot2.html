Webot2

1. 創建剛體球體（Solid 節點）

  • 在場景樹中選擇最後一個節點

  • 按下「Add」新增一個「Solid」節點

  • 在 Solid 節點內添加一個「Shape」節點，並在 Shape 節點中添加「PBRAppearance」節點設定外觀

  • Shape 的 geometry 欄位中添加「Sphere」節點作為幾何體

  • 在 Solid 節點的 boundingObject 欄位中添加一個「Sphere」作為物理邊界

  • 在 Solid 節點的 physics 欄位添加一個「Physics」節點，使物體能受重力影響

  • 將 Solid 節點的位置 (translation) 設置於機器人前方，例如 {0.2, 0, 0.2}

  • 保存並測試模擬，確認球會因重力掉落並可用滑鼠施力移動

2. 優化球體幾何（Sphere 節點）

  • 修改 Sphere 節點的 radius 欄位為 0.05，縮小球體尺寸

  • 將 subdivision 欄位設為 2，提高球體圖形品質

3. 使用 DEF-USE 機制減少冗餘

  • 對第一個 Sphere 節點設置 DEF 名稱為 BALL_GEOMETRY

  • 刪除 boundingObject 中原本的 Sphere 節點

  • 以 USE BALL_GEOMETRY 引用 DEF 節點，讓 boundingObject 與圖形表示共用同一 Sphere

  • 這樣修改 radius 會同時影響兩處，提高維護效率

4. 新增第二顆球

  • 複製第一個 Solid 節點，改名並修改位置避免重疊

  • 在複製節點中，確認 Shape 使用 DEF-USE 機制重用先前定義的球體形狀

  • 調整球的位置參數，使兩球並列展示

5. 建立四面牆壁（Box 節點）

  • 使用 Box 節點定義牆壁的幾何形狀

  • 只使用一個 Shape 節點並以 DEF-USE 機制重用，避免重複定義

  • 將牆壁設定為靜態物體（不加 Physics 節點），使其固定不動

  • 設定牆壁尺寸和位置，形成封閉區域包圍剛體球體

6. 修改牆壁外觀為藍色

  • 在牆壁 Shape 節點的 appearance 欄位添加 PBRAppearance 節點

  • 設定 baseColor 為藍色 (RGB: 0,0,1)

  • 設定 metalness 為 0

  • 設定 roughness 為 0.5

  • 確認所有牆壁共用相同外觀，完成顏色修改

7. 使用 Webots 預設外觀與本地紋理

  • 在球體的 appearance 欄位中刪除原有節點

  • 新增 Webots 預設的 PBRAppearance 節點（如 OldSteel）

  • 添加 ImageTexture 節點並載入本地紅磚牆紋理 red_brick_wall.jpg

  • 確認紋理正確映射在球體上

8. 渲染模式切換與輔助視圖

  • 使用 View > Wireframe Rendering 開啟線框模式，觀察物件網格結構

  • 使用 View > Plain Rendering 回復正常著色模式

  • 使用快捷鍵 Ctrl + F1 顯示座標系統，Ctrl + F10 顯示距離感測器光線

  • 這些工具有助於開發時的觀察與除錯
