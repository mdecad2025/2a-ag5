二、實作步驟詳述

步驟一：啟動 Webots 與建立專案

  1. 啟動 Webots 軟體，首次啟動時選擇任一圖形主題，並關閉導覽教學。

  2. 由選單 File → New Project Directory 建立新專案，專案名稱設定為 my_first_simulation。

  3. 建立新的世界檔，檔名設定為 my_first_simulation.wbt，並勾選「Add a rectangle arena」選項以建立預設競技場。

  4. 將模擬暫停，確認虛擬時間為 0。

步驟二：調整競技場設定

  1. 於場景樹（Scene Tree）中雙擊 RectangleArena 節點。

  2. 將 floorTileSize 由原本的 0.5 0.5 修改為 0.25 0.25。

  3. 將 wallHeight 由 0.1 修改為 0.05，以降低牆壁高度。

  4. 觀察 3D 視窗中地板磁磚與牆壁高度之變化。

步驟三：新增障礙物（木箱）

  1. 選取 RectangleArena 節點並點擊新增按鈕。

  2. 於 PROTO 節點中選擇 objects / factory / containers / WoodenBox (Solid)。

  3. 將木箱尺寸設定為 0.1 × 0.1 × 0.1 公尺。

  4. 調整其 translation 為 0 0 0.05，使箱子正確放置於地板上。

  5. 使用複製與貼上功能建立三個木箱，並將其分別放置於競技場不同位置，避免遮擋中央區域。

步驟四：加入 e-puck 機器人

  1. 於場景樹中點擊新增按鈕。

  2. 選擇 robots / gctronic / e-puck / E-puck (Robot)。

  3. 使用平移與旋轉工具，將 e-puck 機器人放置於競技場中央。

  4. 儲存目前世界檔。

步驟五：建立並撰寫控制程式

  1. 於場景樹中右鍵點擊 e-puck 機器人，選擇新增 Controller。

  2. 控制器語言選擇 Python，檔名命名為 epuck_controller.py。

  3. 於控制程式中撰寫程式碼，使機器人兩側輪子以固定位置轉動，範例如下：

from controller import Robot TIME_STEP = 64 robot = Robot() leftMotor = robot.getDevice('left wheel motor') rightMotor = robot.getDevice('right wheel motor') leftMotor.setPosition(10.0) rightMotor.setPosition(10.0) while robot.step(TIME_STEP) != -1: pass

  1. 儲存程式並執行模擬，觀察 e-puck 機器人向前移動之行為。

步驟六：建立剛體模擬環境

  1. 使用 File → Save World As... 將世界另存為 obstacles.wbt。

  2. 刪除 RectangleArena 節點。

  3. 新增 Floor (Solid) 節點作為地板，並將尺寸設定為 1 m × 1 m。

步驟七：建立球體剛體

  1. 新增一個 Solid 節點作為球體。

  2. 在其 children 欄位中加入 Shape 與 Sphere 節點以定義外觀。

  3. 在 boundingObject 欄位中加入 Sphere 節點作為碰撞邊界。

  4. 在 physics 欄位中加入 Physics 節點，使球體具有物理效果。

  5. 將球體半徑設定為 0.05 公尺，並放置於機器人前方。

步驟八：應用 DEF–USE 機制

  1. 將球體的 Sphere 幾何節點定義為 DEF BALL_GEOMETRY。

  2. 在 boundingObject 中使用 USE BALL_GEOMETRY。

  3. 確保修改一次幾何參數即可同時影響外觀與碰撞邊界。

步驟九：外觀與渲染設定

  1. 為牆壁的 Shape 節點加入 PBRAppearance。

  2. 設定牆壁顏色為藍色，並將 metalness 設為 0、roughness 設為 0.5。

  3. 為球體加入紋理或預設的 PBRAppearance，以提升模擬視覺效果。

  4. 使用 Webots 的線框模式與一般渲染模式，觀察模型結構與外觀差異。
